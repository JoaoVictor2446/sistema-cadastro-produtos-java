# Sistema de Cadastro de Produtos

Este projeto é um sistema simples de cadastro de produtos desenvolvido em Java, com o objetivo de praticar lógica de programação e conceitos básicos de desenvolvimento.

## 📌 Funcionalidades

- Cadastrar produtos
- Listar produtos cadastrados
- Buscar produto por ID
- Remover produto por ID
- Validação de ID único (não permite IDs duplicados)

## 🛠️ Tecnologias utilizadas

- Java
- Programação Orientada a Objetos (POO)
- ArrayList

## ▶️ Como executar

1. Clone o repositório:
```bash
git clone https://github.com/JoaoVictor2446/sistema-cadastro-produtos-java.git
```
2. Abra o projeto em uma IDE (IntelliJ, Eclipse ou VS Code)
3. Execute a classe Main

## 🎯 Objetivo

Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais de Java, como:

- Estruturas de repetição
- Manipulação de listas
- Organização de código em métodos
- Entrada de dados com Scanner
